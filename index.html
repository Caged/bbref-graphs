Basketball Reference Graph Extension

This is a chrome extension for graphing player game log statistics on http://basketball-reference.com. A new button will appear on the right side of the game log tables. Clicking it will allow you to select which statistic you want to graph.

Installation

  • Download For Google Chrome
  • In Chrome, navigate to Window > Extensions
  • Drag and drop the .crx file onto the Extensions page
